# Digital Edge

An MKDocs website for Digital Edge: articles, user guides, how-tos, and browser-based Python playgrounds.

## Run locally

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

Then open `http://127.0.0.1:8000`.

## Build

```bash
mkdocs build
```
